1

2

원커맨드 캠페인

실행 한 번이면 브리프 자동 생성 → 텍스트팀(블로그/SNS/뉴스레터) + 비주얼팀(카드뉴스/썸네일) + 오디오팀(팟캐스트) 3팀이 병렬 작업 → QA 검수까지. 10분 안에 수많은 콘텐츠가 완성됩니다.

3

AI Slop Detection & Fix

다중 LLM 교차 검증으로 AI Slop(AI 특유의 뻔한 표현, 과잉 수식, 반복 패턴)을 자동 감지합니다. 감지된 표현은 즉시 자연스러운 문장으로 재작성. 검색엔진과 독자 모두 속지 않는 품질.

4

수익 피드백 루프

생성 → 업로드 → 성과 수집 → RPM 보정 → 전략 재수립 → 다시 생성. 이 사이클이 자동으로 돌아갑니다. 어떤 키워드가 돈이 되는지, 어떤 포맷이 먹히는지 데이터가 쌓일수록 정확해집니다.

5

중복 방지 벡터DB

Qdrant 벡터DB로 키워드 + 제목 + 본문구조를 3중 검사합니다. 같은 주제를 다른 각도로 써도 구조적으로 겹치면 자동 스킵. 수천 개를 만들어도 하나하나 다른 콘텐츠.

수많은 콘텐츠를 정말 한 번에 만들 수 있나요?
--campaign "AI 자동화" 한 줄이면 브리프 자동 생성 → 3팀(텍스트/비주얼/오디오) 병렬 작업 → QA 검수까지 10분 안에 완료됩니다. 블로그, 쇼츠 스크립트, LinkedIn, Threads, X, 뉴스레터, 카드뉴스(이미지 포함), 팟캐스트(오디오 포함), 썸네일이 한 번에 나옵니다.
알고리즘 역추적이 뭔가요?
7개 플랫폼(네이버/티스토리/유튜브/인스타/LinkedIn/Threads/X)의 인기 콘텐츠를 주기적으로 분석합니다. 어떤 포맷이 노출이 잘 되는지, 몇 시에 올려야 하는지, 하루 몇 개가 최적인지를 데이터로 추적합니다. 알고리즘이 바뀌면 온글이 먼저 감지하고 스케줄을 자동 조정합니다.
AI가 쓴 글이 검색에 노출되나요?
네. 3단계 품질 보장: (1) Multi-pass 검수로 초안→검수→재작성 (2) Claude+Gemini 교차 검증으로 AI Slop 자동 제거 (3) SEO 밀도 재조정 + 내부링크 삽입. 네이버/구글 상위노출에 최적화되어 있습니다.
중복 콘텐츠 문제는 없나요?
Qdrant 벡터DB로 3중 중복 방지합니다. 키워드/제목/본문구조를 각각 벡터로 비교하여, 유사도가 높으면 자동 스킵합니다. 수천 개를 만들어도 모두 다른 콘텐츠입니다.
무료로 시작할 수 있나요?
네. Free 플랜으로 매달 10개 콘텐츠를 무료로 생성할 수 있습니다. 카드 등록 없이 바로 시작하세요. 블로그 생성 + Q&A 시뮬레이션 + 헬스체크 + dry-run이 무료입니다.
수익이 실제로 나나요?
애드센스, 유튜브 광고, 쿠팡파트너스 제휴, 뉴스레터 구독, 전자책 판매, 팟캐스트 스폰서십 — 6가지 수익 채널을 동시에 운영합니다. RPM 기반 수익 트래킹 + 성과 높은 키워드 자동 재투자로 수익이 복리로 성장합니다.
다른 AI 도구 대비 뭐가 다른가요?
대부분의 도구는 "글 생성"까지만 합니다. 온글은 생성 → 중복방지 → AI Slop 제거 → SEO → 알고리즘 맞춤 업로드 → 성과 추적 → 전략 재수립 → 자동 개선까지 전체 사이클이 자동입니다. 이 파이프라인을 직접 구축하면 10개+ 도구에 월 78만원+가 들지만, 온글 하나로 전부 커버됩니다.
시/동화/소설도 만들 수 있다고요?
네. 현대시/자유시/낭만시/산문시 4가지 스타일로 생성됩니다. 한국 현대문학 문체 패턴을 학습한 온글이 운율과 구조를 자동으로 체크합니다. 동화는 연령별(유아/초등/청소년) × 장르별(교훈/판타지/일상/모험) 조합. 소설은 설계서 → 챕터 순차 생성 → 일관성 체크. TTS 낭독 + 영상 합성(MP4)까지 자동 완성됩니다.
